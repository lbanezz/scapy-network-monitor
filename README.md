# 🔐 Network Security Monitor (Scapy + Streamlit)

Ferramenta de monitoramento de rede em tempo real desenvolvida em Python para análise de tráfego e detecção de possíveis ameaças.

---

## 🚀 Funcionalidades

- 📡 Captura de pacotes em tempo real
- 🌐 Identificação de IP de origem e destino
- 📊 Dashboard interativo com Streamlit
- 📈 Gráfico de IPs mais ativos
- 🚨 Detecção de comportamento suspeito (flood simples)
- 💾 Exportação de logs em JSON e CSV
- 🔍 Análise de dados para investigação

---

## 🛠️ Tecnologias utilizadas

- Python
- Scapy
